# 草书的部首

课程目录请看[这里](index.md)。

这里我们系统性地介绍常见的草书部首。有别于简单的字形罗列，我们将部首以互相关联的组为单位进行介绍。

如果您还未学习过[草书字形基础篇](../lessons-basic/README.md)的内容，您可以先系统地学习该内容。如果您想直接学习部首篇的内容，也可以先浏览[本页](../lessons-basic/README.md)的基础知识。这将有助于您更好的理解草书部首的书写逻辑。

草书中的同一部首可能出现在不同字的不同位置。一般来说，只有在特定位置时，它们的写法才与本身不同。因此，我们会重点介绍这些常见的写法不同的草书部首。在其它位置出现时，草书部首与其本身写法类似。

如果您希望快速了解一些部首的草书写法，您可以移步[草书字形结构索引](../dictionary/README.md)，查询一些含该部首的例字。
