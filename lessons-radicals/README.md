## 草书部首的字形变化

草书部首就是一般意义上汉字的部首在草书中的写法。与隶书（或楷书）类似，同一个部首在不同的汉字中可能会呈现出不同的写法（例如楷书中左侧偏旁写法往往略微变形）。在草书中，同一部首的不同写法比隶书（楷书）展现出更丰富的变化。

草书中部首的字形变化往往取决于
* 它在字中的位置、
* 它周围的拥挤程度、
* 它所在字的常见/罕见程度、
* 一些字中的固定写法、
* 艺术需要/避免混淆/书写者选择。

一般来说，同一位置的草书部首如果有多种可能的写法，一般会呈现出“繁复-简化”的区别。选择使用“繁复”或“简化”的写法一般遵照一下原则：
* 选择“繁复”写法：少见的字中、或避免混淆时；
* 选择“简化”写法：在常用字中、或字形拥挤时；

有时也会见到“古-今”的风格区别导致的不同写法。较为古早的章草，后来的今草，以及近现代书法家在一些字的写法上会有所不同。除去一些误写、讹字、以及个别的一些字在个别书法家手中的独特写法，大部分的草书字形还是相通的。随着时代更替，一些讹形和“传统上讲不那么草书的字形”也会取代原有的写法成为通行的写法。

可见草书部首的字形变化十分丰富。一般来说，只有在特定的高频出现的位置时，部首的写法才尤其与其本身写法不同。因此，我们会重点介绍这些特定的部首写法。在其它位置出现时，部首与其本身写法类似。在一些特定组合中出现的固定写法，我们将留到实战篇介绍。
