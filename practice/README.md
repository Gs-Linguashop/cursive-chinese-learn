# 复习字表

一套复习用的字表。定期练习其中的字，熟练掌握，就能自如使用绝大部分的草书字形。


## 常见字（3500个）（字形完善中）

众所周知，因为繁简和异体字的原因，所以最常用3500个字有接近5000个字。但是大部分繁简和异体字的草书是一致的。您可以选取您需要的字练习。
