# 「士」「土」

「士」和「土」书写类似，只在两横长度上有区别。笔顺是「横-竖-横」。

*“吉”*

![吉](../font/MingjianCaoshuHeiti-svg/uni5409.svg)

*“坤”*

![坤](../font/MingjianCaoshuHeiti-svg/uni5764.svg)

「士」字形常常作为「⿱亠丷」的草形或者「大」的省形，在字的上方出现。

*“音”*

![音](../font/MingjianCaoshuHeiti-svg/uni97F3.svg)

在字下方的「土（士）」一般书写成「竖-圈-横」。

*“王”*

![王](../font/MingjianCaoshuHeiti-svg/uni738B.svg)

*“坐”*

![坐](../font/MingjianCaoshuHeiti-svg/uni5750.svg)

# 「大」

「大」与楷书字形差不多。楷书中的大字形在草书中也一般写作「大」形

*“大”*

![大](../font/MingjianCaoshuHeiti-svg/uni5927.svg)

「大」在字上方可以简化成「士」。

*“奇”*

![奇](../font/MingjianCaoshuHeiti-svg/uni5947.svg)

「大」在字上方，有时会简化成类似「反犬旁」的「撇」起笔，因为这样可以少一次抬笔。

*“奄”*

![奄](../font/MingjianCaoshuHeiti-svg/uni5944.svg)

「大」在字的下方，经常作为「貝贝」字旁的省略出现。

*“賣卖”*

![賣卖](../font/MingjianCaoshuHeiti-svg/uni8CE3.svg)

「大」在字的下方，经常作为「廾」字旁的省略出现。

*“弄”*

![弄](../font/MingjianCaoshuHeiti-svg/uni5F04.svg)

「美字底」较少出现。

*“美”*

![美](../font/MingjianCaoshuHeiti-svg/uni7F8E.svg)

# 「己」

「己」「已」「巳」「㔾」在草书中几乎没有区别，都是由「𠃌-乚」构成。

*“卷”*

![卷](../font/MingjianCaoshuHeiti-svg/uni5377.svg)

「己」形经常作为「豆」字的省略出现，也可以进一步省略写成「三」。

*“豈岂”*

![豈岂](../font/MingjianCaoshuHeiti-svg/uni8C48.svg)

# 「𭕄」「三点底」

横向并列的重复形状常常被省略为「𭕄」。

*“僉佥”*

![僉佥](../font/MingjianCaoshuHeiti-svg/uni50C9.svg)

字下方的「灬」「心」「川」常常被写作「三点底」。「三点底」进一步简化可以写作一横。

# 「横弯」「卩」「々」「⻏」

「横弯」经常作为各种如「冖」「門门」偏旁的草书写法。

*“間间”*

![間间](../font/MingjianCaoshuHeiti-svg/uni9593.svg)

「卩」写作「横弯-点」。

*“卻却”*

![卻却](../font/MingjianCaoshuHeiti-svg/uni5374.svg)

「々」常用作书写「撇」起笔的，含包围形状的右侧部首，比如右侧的「月」「欠」。

*“有”*

![有](../font/MingjianCaoshuHeiti-svg/uni6709.svg)

*“吹”*

![吹](../font/MingjianCaoshuHeiti-svg/uni5439.svg)

「阝（右）」常用作书写「横」起笔的，含包围形状的右侧部首，比如右侧的「阝（右）」「頁页」。在实际书写时，与「々」几乎没有区别。

*“那”*

![那](../font/MingjianCaoshuHeiti-svg/uni90A3.svg)
